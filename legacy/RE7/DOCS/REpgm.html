The current version is v5.2 10 Oct 1997

Program Requirements

In order to run RE, you'll need to get the following list of things together in the same place at the same time:

  • The program and supporting files. This may be downloaded as a compressed file (.ZIP) and decompressed the usual way with WINZIP or PKUNZIP.

    Download RE52.ZIP (890K)

    If you have trouble downloading it, please contact us, we will send it to you by email or send you a floppy.

  • A computer capable of running or emulating MS-DOS version 3.1 or higher. This includes computers running Window 3.1 (or higher) Windows 95, or Windows NT. Besides being able to accomplish this feat, your computer must also
    • Be able to run 80386 software in protected mode. Basically, this means you need an 80386 processor or higher. Almost any Windows machine qualifies.
    • There is no precise way to say how much memory is needed for any particular set of data, but 8 megabytes seems to be fine for most applications. For the Tamang data with which we have been experimenting, 8 megs has been quite adequate. A statistical summary of this data (for planning purposes only) is included in Appendix E.
    • Have approximately three megabytes of disk space available for the RE program and supporting files. You may need substantially more if you have large data files!
    • Support ANSI.SYS, which is used by the windowing functions of RE. You may have to install this in your CONFIG.SYS file, though in some computers this support is built-in. See your computer's hardware manual for instructions. It is automatically part of most Windows setups.
