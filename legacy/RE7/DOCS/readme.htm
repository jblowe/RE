A. DESCRIPTION OF DATA SETS

The sets of example files provided with the RE program are organized in subdirectories beneath the PRJ directory. The set of example files are supplied with the program illustrates a number of linguistics and computational features of the program.

Six data sets are supplied with the program. These are listed below in approximate order of size and complexity:

  • EXER   A tiny table of made-up correspondences designed to exercise basic features of the program: interactive upstream and downstream computations, cognate set creation, and syllable structure.
  • POLY   A tabular set of 36 words in four polynesian languages (Tongan, Samoan, Rarotoan, and Hawaiian) (from xxx 19xx:999), with table of correspondences. This data set illustrates "batch" upstream processing (i.e. reconstructing ancestors and creating cognate sets on the basis of modern forms) and "recursive" syllable processing.
  • DIS  A small subset (7 selected words in 8 languages) of the TGTM database which illustrates upstream processing of heterogenous lexicons (dictionaries) of different languages (i.e. multi-file processing), handling of isolates, consolidation of cognate subsets based on membership.
  • DEMO  A larger TGTM dataset, containing words from 8 modern dialects, forming 69 cognate sets, based on the data and reconstructions presented in Mazaudon 1978. Illustrates the phonological feature processing abilities of RE, the use of semantic constraints (MELs) and includes a substantially complete table of correspondences.
  • LOLOISH Contains substantial datasets and partial analysis for two subgroups of Tibeto-Burman (Northern Loloish and Southern Loloish) based on work presented in Hansson 1985 and Chen 1986. The languages represented are closely related and show how RE can be used in microlinguistics studies.
  • IERE  A small sets of Germanic verbs prepared using data given in Hoch 198x. The electronic version of this dataset was created by Collin Baker, UC Berkeley.
